Exporting the secret risk assessment report to CSV

Export the secret risk assessment report to a CSV file for detailed investigation and stakeholder sharing.

  1. On GitHub, navigate to the main page of the organization.
  2. Under your organization name, click the Security and quality tab.
  3. In the sidebar, under "Security", click Assessments.
  4. Towards the top-right side of the report, select the dropdown menu, then click Download CSV.
